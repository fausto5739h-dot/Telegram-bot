# Telegram Bot Project

## Description
This project is a Telegram bot that helps users with various tasks, including automation, reminders, and information retrieval.

## Features
- **Automation**: Automated responses for common queries.
- **Reminders**: Users can set reminders.
- **Information Retrieval**: Get information from APIs and other sources.

## Installation
To install this bot, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/fausto5739h-dot/Telegram-bot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Telegram-bot
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To run the bot, use the following command:
```bash
node bot.js
```

## Configuration
- Create a `.env` file and add your Telegram Bot API key:
  ```plaintext
  TELEGRAM_BOT_TOKEN=your_bot_token_here
  ```
